Project Ink - Testing Document
==============================

Welcome to the test file for Ink, a minimalist terminal pager.
This file is designed to test various features of the pager, including:
1. Smart word-wrapping at word boundaries (spaces and hyphens).
2. Dynamic 8% side margins.
3. Smooth scrolling and navigation.
4. Regex-based searching.
5. Responsive terminal resizing.
